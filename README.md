# RESTBlog
**A MEN based REST routes obeying blog**

Prerequisites
-------------
* MongoDB
* nodejs
* npm

Usage
------------
> Clone the repository

> npm install

> Run mongodb daemon

> Provide enviornment variables

> node app.js

Features :
----------
* Add a new blog
* Update an existing blog
* Delete a blog
